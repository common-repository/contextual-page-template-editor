Edit page templates from within the page they are being used for

== Description ==

With this wordpress plugin you can edit page templates from within the page they are being used for, you will have a new textarea to edit the actual .php file that is the page template that the page you’re editing is currently using. If it is not using any custom page template, it doesn’t show the textarea.

[Plugin Author homepage](http://www.noextratime.com/wordpress-plugin-contextual-page-template-editor/ )
== Installation ==

1. Unzip `contextual-template-editor.zip` and Upload to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to edit one of your your pages that uses a page template and have fun!
